# Project - Sakhi- An AI Intelligence Chatbot for Supporting Workplace Harassment Victims
The "Sakhi" system aims to reduce psychological distress experienced by survivors of workplace sexual harassment by using a chatbot to provide advice in compliance with the Prevention of Sexual Harassment (POSH) Act of India. The system includes three artificial intelligence components: spatio-temporal information extraction, harassment type identification, and user conversation. The unique feature of offering legal advice specific to Indian law sets it apart from existing chatbots.
